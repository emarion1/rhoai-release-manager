🔄 Release Cycle Structure

Each RHOAI release has 3 events delivered quarterly:

  • EA1: Early Access 1 (DP/TP features only)
  • EA2: Early Access 2 (DP/TP features only)
  • GA: General Availability (DP/TP/GA features)

Maturity Levels: DP (Dev Preview) → TP (Tech Preview) → GA (General Availability)

📏 Capacity Guidelines

  • 🟢 Conservative: ≤30 pts
  • 🟡 Typical: 30-50 pts
  • 🟠 Aggressive: 50-80 pts
  • 🔴 Over Capacity: >80 pts

Historical baseline: 27.5 pts/event median

🏷️ Feature Status

  • Fix Version: Committed and approved
  • Target Version: Intended but not committed
  • In Plan: Ranked in JIRA Advanced Roadmaps Plan
  • Not in Plan: In RHAISTRAT project but not in plan

💡 Tips

  • Click ℹ️ icons throughout the interface for contextual help
  • Features show story points - auto-sized if not set in JIRA
  • Drag features between events to rebalance capacity
  • Watch capacity meters - stay under 80 pts per event
